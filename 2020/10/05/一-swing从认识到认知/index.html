发表于2020-10-05|更新于2020-10-05

学习初衷

最近计算机网络课程设计就要开始动手做了，在确定了需求之后，决定使用java进行开发。涉及到java可视化操作，在查阅资料之后，发现awt和swing是比较常用的java可视化开发程序包。但是awt是一个抽象窗口工具包，早期编写图形界面应用程序的包。swing是为解决 AWT 存在的问题而新开发的图形界面包。Swing是对AWT的改良和扩展。以前也略有涉及swing的使用，但是因为没有系统的学习，现在很多知识点都不够牢固。所以还是要从0开始重新学习swing的使用。

关于swing

  1. Swing 是 Java 为图形界面应用开发提供的一组工具包，是 Java 基础类的一部分。

  2. Swing 包含了构建图形界面（GUI）的各种组件，如: 窗口、标签、按钮、文本框等。

  3. Swing 提供了许多比 AWT 更好的屏幕显示元素，使用纯 Java 实现，能够更好的兼容跨平台运行。

  4. 为了和 AWT 组件区分，Swing 组件在javax.swing.*包下，类名均以 J 开头，例如: JFrame、JLabel、JButton等。

Swing组件

一个 Java 的图形界面，由各种不同类型的“元素”组成，例如: 窗口、菜单栏、对话框、标签、按钮、文本框等等，这些“元素”统一被称为 组件（Component）。
组件按照不同的功能，可分为 顶层容器、中间容器、基本组件。一个简单窗口的组成，如下层级结构所示:

  • 顶层容器
    • 菜单栏
    • 中间容器
    • 特殊容器
      • 基本组件
      • 不可编辑信息的显示组件
      • 可编辑信息的显示组件
      • 特殊对话框组件

组件类型的继承关系:

  1. 顶层容器 属于窗口类组件，继承自java.awt.Window；
  2. 中间容器 和 基本组件 继承自javax.swing.JComponent。
